Music of Minecraft Extension

Options


Basic settings




The volume at which to play the music.

Each new track is delayed by a random time interval. This is limited to avoid too long a wait.


Music source



This should be the Minecraft assets folder, which should be present if you have Minecraft installed on your computer. Consult the table below to find it on your system. When asked to confirm uploading of many files to this site, accept it. This is necessary for the extension to access the folder, so music files can be extracted. NOTE: you do not need to find the audio files - they are hashed so they don't look like audio files, but the extension can find them if you choose the entire assets directory.
Operating System Minecraft assets path
Windows C:/Users/{USERNAME}/AppData/Roaming/.minecraft/assets/
Mac ~/Library/Application Support/minecraft/assets/
Linux ~/.minecraft/assets/


This should be a folder containing music files to play. Music files should be of types: .mp3, .ogg, .wav, .mp4, .m4a, .ogv, .ogm, .oga, .opus, .webm, .flac. All subdirectories will also be scanned for music files. Note that changes to the contents of this folder will not be registered unless it is re-uploaded here.
